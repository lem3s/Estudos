Sistema de Banco para o usuário final


Olá professora, para essa parte inicial do projeto fizemos a definição dos tipos de dados
e a inicialização das estruturas fazendo a parte de persistência de dados usando um arquivo csv
com um proposta de execução de um aplicativo de banco para o usuário final (correntista) que
poderá criar uma conta, acessar a conta, verificar o saldo, fazer transferências, entre outros.

Para rodar o programa basta usar o comando "make" no diretório contendo os arquivos ou compilar
normalmente o arquivo main.c e executar o binário compilado.
